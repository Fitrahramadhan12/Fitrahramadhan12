Cara Menampilkan Data dalam JavaScript

Terdapat beberapa cara untuk menampilkan data dalam JavaScript:

  1. innerHTML - Metode ini digunakan untuk menampilkan data ke dalam elemen HTML.
    Contoh: document.getElementById("demo").innerHTML = "Hello World";
  2. document.write() - Digunakan untuk menulis langsung ke dokumen HTML.
    Namun, tidak disarankan digunakan setelah dokumen HTML dimuat karena akan menimpa seluruh konten yang ada.
  3. console.log() - Metode ini digunakan untuk menampilkan data di console browser,
    sangat berguna untuk debugging dan pengembangan.
  4. alert() - Menampilkan data dalam bentuk pop-up box/dialog box.
    Contoh: alert("Hello World!");
